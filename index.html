Git é um sistema de controle de versão distribuído livre e de código aberto. Criado em 2005 por Linus Torvalds.
Vantagens:
* Reverter o código
* Descobrir quem modificou o código
* Trabalho Offline

Download e Instalação:
Download GIT: http://git-scm.com/download/
Commande line GIT:
GIT: $ sudo apt-get install git
Commande line GIT (Interface Gráfica):
GITG: $ sudo apt-get install gitk
GITK: $ sudo apt-get install gitg

Configurando

Configurando pelo terminal:
$ git config --global color.status auto
$ git config --global color.branch auto
$ git config --global color.diff auto
$ git config --global color.ui always
$ git config --global core.editor vim
$ git config --global user.name "meunome"
$ git config --global user.email "eu@example.com"
Configurando pelo editor de texto:
- encontre o arquivo .gitconfig na sua pasta home e abra e modifique
[user]
name = Walter Nascimento
email = walter@m2smart.com.br
[ui]
color = always
[color]
ui = auto
status = auto
diff = auto
[core]
editor = vim

Iniciando Projeto

Criando repositório
$ git init "diretório"
$ cd "diretório"

Clonando um Repositório
$ git clone git@github.com:walter/it.git
Pratica:
1 Crie um arquivo ou edite
2 $ git add "arquivo" = adiciona o arquivo
3 $ git commit -s -m "[ACAO] Seu comentario"
4 $ git fetch = atualizar
4.1 $ git rebase origin/master = junta o arquivo baixado com o repositorio local
5 $ git push = envia arquivo
Alguns comandos Extras:
$ git reset --soft HEAD~1 = reverte
$ git checkout . = restaura um arquivo
$ git diff . = mostra as alterações
$ git rebase --continue = continua o rebase
$ git stash = cria um branch temporario
$ git blame . = ver quem alterou
$ git commit --amend (junta o comite)
$ git reset HEAD . (unstage)
